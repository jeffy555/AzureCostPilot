# Terraform Child Modules

This repository contains reusable Terraform child modules for managing cloud infrastructure.

## Structure

Each folder represents a separate child module for a specific resource type:

- `ResourceGroup/main.tf`
- `ResourceGroup/variables.tf`
- `ResourceGroup/outputs.tf`
- `StorageAccount/main.tf`
- `StorageAccount/variables.tf`
- `StorageAccount/outputs.tf`

## Configuration

- **Cloud Provider**: AZURE
- **Module Approach**: Child Module (reusable component)

## Usage

These child modules can be called from a parent/root module:

```hcl
module "example" {
  source = "./ModuleName"
  
  # Pass required variables
  name     = "example"
  location = "eastus"
}
```
